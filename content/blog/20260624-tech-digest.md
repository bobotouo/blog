1. **构建本地 LLM 可视化调试环境**：基于 Token 与向量化文章，实现代码级别的输入输出追踪，帮助团队快速定位模型误差。

4. **大文件抓包与监控**：在现有 API 网关引入 ApiCatcher 思路，实现对关键业务大文件上传/下载的实时抓包与性能分析。
