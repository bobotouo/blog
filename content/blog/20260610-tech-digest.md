- **大模型编排**：AgentSandbox 与 Peri Code 展示了多模块、工具层级的编排方案。

- **AI 与音频安全**：Linux 6.2 引入低延迟、零信任音频安全架构，为 AI 音频应用提供底层保障。  
- **全栈实战**：Node.js + LangChain + React 实现个人知识库，涵盖后端、前端与移动端。  
- **前端工程化**：TypeScript 项目配置、路径别名与 ESM 兼容性最佳实践。  
- **Flutter 与 AI 交互**：flutter_agent_lens 通过 MCP 服务把 DevTools 暴露给 AI，开启跨语言调试新模式。
